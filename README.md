
Sleep Debt is an iOS app that helps users understand their sleep habits using Apple Health data.

## Features
- Apple Health sleep tracking
- Sleep trend analysis
- Smart reminders
- AI-powered premium insights
- Clean dark-mode UI

## Tech Stack
- Flutter
- Dart
- Apple HealthKit
- In-App Purchases
- AI-based insight flow
